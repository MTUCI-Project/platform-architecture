# Хранилище файлов

Для хранения медиа используется MinIO.

Структура бакетов:

tracks/

covers/

tts/

recordings/

---

User Backend

Использует MinIO для загрузки файлов.

---

Radio Engine

Читает аудиофайлы непосредственно из MinIO.

---

AI Backend

Сохраняет сгенерированные TTS-файлы в MinIO.

---

Никакие файлы не должны храниться внутри контейнеров сервисов.

Контейнеры считаются полностью статeless.
